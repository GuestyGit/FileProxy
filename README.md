# FileProxy

FileProxy is a proxy that is basically a html file


## Authors

- [@GuestyGit](https://www.github.com/GuestyGit)


## Deployment

To deploy this project run

```bash
$ git clone https://github.com/GuestyGit/FileProxy --recursive
$ cd FileProxy
$ npm install
$ npm start
```
---


## Startup

To run the project use

```bash
  npm start
```
